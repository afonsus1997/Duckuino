## PINMAP
PINMAP of the ATMEGA32u4

| PIN | ALT FUNCTION | OLED | Thumbwheel | LEDs | DIP Sw's |       SD       |
|:---:|:------------:|:----:|:----------:|:----:|:--------:|:--------------:|
| D2  | SDA          |  SDA |      -     |   -  |     -    |        -       |
| D3  | SCL          |  SCL |      -     |   -  |     -    |        -       |
| D4  | A6           |   -  |      -     |   -  |     -    | CD/DAT3 (!CS!) |
| D5  | -            |   -  |      -     |   -  |     4    |        -       |
| D6  | A7           |   -  |      -     | LED1 |     -    |        -       |
| D7  | -            |   -  |      -     | LED2 |     -    |        -       |
| D8  | A8           |   -  |   Center   |   -  |     -    |        -       |
| D9  | A9           |   -  |      -     |   -  | 1 (Boot) |        -       |
| D10 | A10          |   -  |      -     |   -  |     2    |        -       |
| D11 | -            |   -  |    Right   |   -  |     -    |        -       |
| D12 | -            |   -  |    Left    |   -  |     -    |        -       |
| D13 | -            |   -  |      -     |   -  |     3    |        -       |
| D14 | -            |   -  |      -     |   -  |     -    |    DAT0 (D0)   |
| D15 | -            |   -  |      -     |   -  |     -    |   CLK (SCLK)   |
| D16 | -            |   -  |      -     |   -  |     -    |    CMD (DI)    |
